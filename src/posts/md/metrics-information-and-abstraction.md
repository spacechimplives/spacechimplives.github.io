So when dealing with Euclidian space, we're talking about degrees of freedom that are continuous and perpendicular.  When we switch to relativistic hyperbolic space, this is still the case, however, the sign of one element of the metric signature is switched.

One problem with the way we view space is that there is no way for it to be hierarchical.  If we're going to treat information states as points in some coordinate system, then we've got to be able to filter out the commonalities so that we're not adding redundancy to the system when we say that an object is an animal as well as a frog.  Since frogs are animals, the state of being a frog is not perpendicular, or totally uncorrelated, to the state of being an animal.  If you can view the state of being a frog as a continuous dimension (easy for a cognitive scientist--see family resemblances--but will deal with this topic in [this future post](http://spacechimplifecom.local/2017/12/29/categorization-from-binomial-to-gaussian/)), then you can see that movement along the dimension of being a frog will have some correlation with movement along the dimension of being an animal.
