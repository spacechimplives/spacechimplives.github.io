I’m attempting to build software for this purpose.  I need funding because it’s ambitious, but it’s at a pretty reasonable scale compared to the complexity of the problem, and it should be profitable, so should be able to self-fund later and give a good return.  It’s a pretty simple idea: (1) create a language which can unify written policy and machine code (2) use that code to encode policies for various institutions (3) use a pr process to allow people to directly submit their ideas (4) others up-vote or down-vote Reddit style. (5) policies which are approved are now integrated into the policy-base.

Start this with very small institutions… freelancers collaborating, roommates, clubs, small companies, HOA’s.  Grow it to medium-sized companies, local councils, etc.  Eventually it the basis of a direct democracy party where candidates are expected to propose things that get voted up etc.  eventually maybe it can be used for some local governments, and suddenly rather than people contributing a few bits every few years to the decision-making process, people can contribute daily.  That is information from the ground directly into the decision-making apparatus   That *will* outcompete any existing system in the long run.
