This is purported to be some MIT professor's thoughts on Economics: http://www.math.ist.utl.pt/~sjh/thoughts

Now there is some unnecessary ad hominem in there, as well as some misunderstanding of economics, but I find a point to be particularly accurate: the fact that economics purports to describe reality, but can get absurdly far away from it.  Another major problem with economics is that is creates itself.  The economic models have the power to drive human behavior, and in doing so reinforce the models that they have created.  I don't think this is particular to economics, but it is where it occurs most obviously.
