- Behavior

 	generation/execution of code by an agent/executor

 	which results in an increase of reachable states



 	Following definitions inspired by "Representation and Behavior" & "Incomplete Nature"
 	Behavior

 	- Action which is constrained by internal state(s) of an agent/executor, which is anticipated by the agent to move the internal state of an agent/executor toward an internal equilibrium

 	possibly drives the system further from equilibrium in an attempt to reach final equilibrium

 	failure to reach equilibrium results in further instructions which anticipate same equilibrium



 	equilibrium states:

 	- states in which less is being anticipated?

 	instructions quickly result in anticipated response

 	for humans: dopamine recently received

 	- states which result in a lower complexity or lower amount of code being executed

 	a shorter call stack







 	for turing machine: states that lead to halting






 	Intentional Behavior

 	- Behavior which is executed involving the agent's representations of the environment and/or self

 	Representation: internal state that is anticipated to be correlated with external state such that when that correlation is reduced, another internal state is moved out of equilibrium, triggering behavior to reinstate the correlation





 	Conscious Behavior

 	- Intentional behavior which is executed involving the agent's internal states which represent itself







 	- Institution

 	A group of humans

 	which interacts with a set of defined rules

 	Rules: actions or prohibitions that are expected of members in response to a given context

 	(optional in theory, required in practice) the ability to distinguish whether the rule has been followed

 	therefore: in practice, equivalent to instructions



 	Rules can trigger more rules, resulting in the equivalent of a call stack
 	Rules can be more broadly defined as constraints on actions that members may take

 	- i.e. constraints on behavior...

 	increases probability of some behaviors, decreases probability of others.

 	- requires physical work to change equilibrium
